# wxve-chat

Open-source chat interface for [Xve](https://chat.wxve.io), the analytical voice of Wxve.

This also serves as a reference implementation for integrating with the Wxve chat API.

## Quick Start

```bash
# Install dependencies
cargo install trunk
rustup target add wasm32-unknown-unknown

# Run locally
trunk serve
```
